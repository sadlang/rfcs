# المقترحات المعتمَدة (Accepted RFCs)

يحوي هذا المجلد المقترحات (RFCs) التي **تمّ اعتمادها** للغة ص.

- كل ملف يُسمّى بالنمط: `NNNN-اسم-وصفي.md` حيث `NNNN` هو رقم الـ RFC (رقم الـ PR
  الذي دُمج عبره).
- وجود مقترح هنا يعني الموافقة على **مبدئه**، لا بالضرورة اكتمال تنفيذه.
- لمتابعة حالة التنفيذ، راجع Issue التتبُّع المرتبط بكل مقترح.

لكتابة مقترح جديد، ابدأ من القالب [`0000-template.md`](../0000-template.md) في جذر
المستودع، واتبع [العملية الموضّحة في README](../README.md).
